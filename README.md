# CreatAdventure
CreatAdventure é um jogo em 2D com gráficos em pixel que segue a jornada épica de Kairos, um jovem artista em busca de recuperar sua criatividade e salvar a criatividade do mundo. Enfrente desafios e batalhe contra o Tiranus da Criatividade Bloqueada para libertar a arte e a inspiração
